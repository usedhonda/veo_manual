# キャラクターシート作成ガイド

## 概要

キャラクターシートは、Veo 3.1のIngredientsモードで使用する参照画像です。複数のシーンでキャラクターの一貫性を保つために不可欠です。

**目的**:
- キャラクターの外見を固定
- 複数シーンで同じキャラを使用
- アニメ/スタイライズ作品での一貫性

---

## キャラクターシートの構成

### 理想的なシート構成

```
┌─────────────────────────────────────────────────────────────┐
│                    キャラクターシート                        │
│                                                             │
│  ┌───────┐  ┌───────┐  ┌───────┐  ┌───────┐               │
│  │ 正面  │  │  45度 │  │ 横顔  │  │ 後ろ  │               │
│  │       │  │       │  │       │  │       │               │
│  └───────┘  └───────┘  └───────┘  └───────┘               │
│                                                             │
│  ┌───────┐  ┌───────┐  ┌───────┐                          │
│  │ 笑顔  │  │ 真剣  │  │ 驚き  │  表情バリエーション       │
│  └───────┘  └───────┘  └───────┘                          │
│                                                             │
│  カラーパレット: ████ ████ ████                            │
└─────────────────────────────────────────────────────────────┘
```

### 必須要素

| 要素 | 重要度 | 説明 |
|------|--------|------|
| 正面ビュー | ★★★ | 顔の特徴を明確に |
| 45度ビュー | ★★★ | 立体感の把握 |
| カラーパレット | ★★☆ | 色の一貫性 |
| 表情バリエーション | ★★☆ | 感情表現用 |
| 横顔 | ★☆☆ | 横からのショット用 |
| 後ろ姿 | ★☆☆ | 追従ショット用 |

## 作成手順

### Step 1: キャラクター設計

```
【基本情報】
- 名前/仮名: [識別用]
- 年齢/見た目: [年代]
- 性別: [男性/女性/その他]
- 性格: [外見に影響する要素]

【ビジュアル設計】
髪:
- 色: [具体的な色名 + HEXコード推奨]
- スタイル: [長さ、形状]
- 特徴: [髪飾り、アクセサリー]

目:
- 色: [瞳の色]
- 形: [丸い/鋭い/たれ目/つり目]
- 特徴: [ハイライト、まつげ]

服装:
- ベース: [制服/カジュアル/フォーマル]
- カラー: [主色、アクセント色]
- 特徴: [ロゴ、装飾]

アクセサリー:
- [リスト形式で列挙]
```

### Step 2: スタイル選択

| スタイル | 特徴 | 適したコンテンツ |
|---------|------|------------------|
| 現代アニメ | クリーンな線、大きな目 | MV、ショートアニメ |
| 少年漫画風 | ダイナミック、影強め | アクション系 |
| 少女漫画風 | ソフト、キラキラ | ロマンス、日常 |
| ユーフォテーブル風 | エフェクト重視 | バトル、ファンタジー |
| 新海誠風 | 背景リアル、光の表現 | ドラマ、風景重視 |

### Step 3: Geminiへの依頼

```
Veo 3.1のIngredients用キャラクターシートを生成してください。

【キャラクター情報】
- 髪: [色、スタイル、特徴]
- 目: [色、形]
- 服装: [詳細な説明]
- アクセサリー: [リスト]

【シート要件】
- アングル: 正面、45度、横顔
- 表情: 笑顔、真剣、驚き
- 背景: シンプルな白またはグレー
- スタイル: [選択したスタイル]

【技術仕様】
- 解像度: 1080p相当
- アスペクト比: 16:9
- 用途: Ingredients参照画像

【ネガティブ】
- 複雑な背景
- バラバラなスタイル
- 不鮮明な線
```

## 具体的な依頼例

### 例1: 学園系女子キャラ

```
【キャラクター】
- 髪: ターコイズブルーのツインテール（腰まで）
- 目: 同じターコイズブルー、大きめ
- 服: セーラー服（白ブラウス、紺スカート、赤リボン）
- アクセサリー: 黒いヘアリボン×2

【シート要件】
- 3アングル（正面、45度右、横顔左）
- 3表情（笑顔、真剣、照れ）
- 全身が見える構図
- 背景: 白

【スタイル】
- modern anime
- clean lines
- cel-shaded look
```

### 例2: ファンタジー剣士

```
【キャラクター】
- 髪: 銀髪のショート、サイドに流れる
- 目: 琥珀色、鋭い
- 服: 黒の革鎧、青いマント
- 武器: 細身の剣（腰に）
- アクセサリー: 銀のペンダント

【シート要件】
- 4アングル（正面、左45度、右45度、後ろ）
- 2表情（冷静、戦闘時）
- 上半身メイン
- 背景: ダークグレー

【スタイル】
- dark fantasy anime
- dramatic shading
- detailed armor
```

### 例3: 現代カジュアル男性

```
【キャラクター】
- 髪: 黒髪、ナチュラルな短髪
- 目: 茶色、穏やか
- 服: 白Tシャツ、デニムジャケット
- アクセサリー: シンプルな腕時計

【シート要件】
- 3アングル（正面、45度、横）
- 3表情（微笑み、笑い、考え込む）
- 上半身
- 背景: ライトグレー

【スタイル】
- slice of life anime
- soft shading
- natural proportions
```

## Veo 3.1での使用方法

### Ingredientsモードでの指定

```
【参照画像の使用】
- キャラクターシート画像を指定
- referenceType: "asset"（重要）
- referenceType: "style"は3.1では非対応

【プロンプトでの補強】
毎回キャラの特徴を再記述:
"Using character from reference image:
turquoise twin-tail hair with black ribbons,
turquoise eyes, sailor uniform (white, navy, red)"
```

### 一貫性を保つコツ

```
1. 同じ参照画像を全シーンで使用
2. プロンプトでも特徴を毎回記述
3. ネガティブに追加:
   - "off-model"
   - "different hairstyle"
   - "wrong hair color"
   - "costume change"
```

## よくある問題と対策

### 問題1: シートの画像がバラバラ

**原因**: スタイルの指定が弱い

**対策**:
- スタイルを詳細に指定
- 「consistent style across all views」を追加
- 一度に生成（分割生成しない）

### 問題2: 色が正確に出ない

**原因**: 色の指定が曖昧

**対策**:
- HEXコードを含める: `turquoise (#00CED1)`
- 「exact color」を強調
- カラーパレットセクションを追加

### 問題3: 服装のディテールが変わる

**原因**: 説明が不足

**対策**:
- 服のパーツごとに詳細記述
- 「consistent costume design」を追加
- 特徴的なディテールを強調
