# 背景・環境画像ガイド

## 概要

シーンの背景や環境を構成する画像を生成するガイドです。

**用途**:
- シーンの雰囲気設定
- Text-to-Videoの参考
- First Frameとしての環境ショット
- ループ素材の開始点

---

## 背景の種類

### 自然環境

| 環境 | 特徴 | 適した用途 |
|------|------|-----------|
| 森林 | 緑、木漏れ日、深み | ファンタジー、癒し |
| 海岸 | 青、波、広がり | 爽快感、開放感 |
| 山岳 | 壮大、雪、雲 | 壮大さ、達成感 |
| 草原 | 黄金、風、広大 | ノスタルジー、自由 |
| 砂漠 | オレンジ、砂丘、夕日 | 孤独、冒険 |

### 都市環境

| 環境 | 特徴 | 適した用途 |
|------|------|-----------|
| 現代都市 | ビル、車、人々 | ビジネス、現代劇 |
| 夜の街 | ネオン、反射、雨 | サイバーパンク、ノワール |
| 下町 | 路地、看板、生活感 | 日常、人情 |
| 未来都市 | ホログラム、浮遊、光 | SF、テクノロジー |

### 室内環境

| 環境 | 特徴 | 適した用途 |
|------|------|-----------|
| オフィス | クリーン、モダン、窓 | ビジネス、CM |
| 住宅 | 暖かみ、家具、光 | 日常、家族 |
| スタジオ | 照明、機材、シンプル | 撮影、アート |
| カフェ | コーヒー、木目、人々 | カジュアル、出会い |

## 時間帯別の演出

### ゴールデンアワー（黄金時間）

```
【特徴】
- 日の出/日没前後の約1時間
- 暖かいオレンジ〜ゴールドの光
- 長い影、ソフトな質感

【指定方法】
"golden hour, warm sunlight, long shadows,
soft golden glow on surfaces"

【適した用途】
- ロマンティックなシーン
- ポートレート
- 感動的な瞬間
```

### ブルーアワー（青の時間）

```
【特徴】
- 日没後/日の出前の薄明
- 冷たい青〜紫の光
- 人工光とのコントラスト

【指定方法】
"blue hour, cool twilight, city lights emerging,
deep blue sky gradient"

【適した用途】
- 都市夜景
- ミステリアスな雰囲気
- 切なさの演出
```

### 真昼

```
【特徴】
- 強い直射光
- 短い影、高コントラスト
- クリアな色彩

【指定方法】
"midday sun, harsh shadows, vibrant colors,
clear blue sky"

【適した用途】
- アクションシーン
- ドキュメンタリー風
- エネルギッシュな場面
```

### 夜

```
【特徴】
- 人工光源（街灯、ネオン）
- 暗い領域と明るいスポット
- 高コントラスト

【指定方法】
"nighttime, artificial lighting, neon signs,
wet pavement reflections"

【適した用途】
- ノワール
- ナイトライフ
- ドラマチックなシーン
```

## 天候の表現

### 晴れ

```
【指定】
"clear sky, bright sunlight, crisp shadows"

【動画での効果】
- 安定した光
- ポジティブな印象
- クリアな視界
```

### 曇り

```
【指定】
"overcast sky, soft diffused light, no harsh shadows"

【動画での効果】
- 柔らかい光
- 落ち着いた雰囲気
- 色が均一に出る
```

### 雨

```
【指定】
"rainy weather, wet surfaces, reflections on pavement,
visible raindrops, gloomy atmosphere"

【動画での効果】
- 反射と動き
- メランコリックな雰囲気
- 視覚的に豊か
```

### 霧

```
【指定】
"foggy atmosphere, limited visibility,
silhouettes in mist, mysterious mood"

【動画での効果】
- 深度感
- 神秘的
- ソフトな背景
```

### 雪

```
【指定】
"snowy landscape, white covering, falling snowflakes,
cold winter atmosphere"

【動画での効果】
- 静寂感
- 季節感
- 美しい光の反射
```

## 動画用の背景設計

### パン/トラッキング用

```
【要件】
- 横に広い構図
- 連続性のある背景
- 視線を誘導する要素

【例】
街路：建物が連続、歩道、街灯の列
森林：木々が続く、小道、光の筋
海岸：波打ち際が続く、砂浜、遠くの岬
```

### ドリー用

```
【要件】
- 奥行きのある構図
- レイヤー感（前景・中景・背景）
- 焦点となるポイント

【例】
森の道：手前の木→道→奥の光
オフィス：手前のデスク→通路→窓
街路：手前の人→道→遠くのビル
```

### オービット用

```
【要件】
- 中央の被写体を囲む環境
- 360度どこから見ても成立
- 均一な環境光

【例】
スタジオ：被写体を中心に均一な背景
自然：木々に囲まれた空間
室内：部屋の中央
```

### ループ素材用

```
【要件】
- 始点と終点が繋がる
- 自然な繰り返し要素
- 単純な動きのパターン

【例】
炎：揺らめく炎
水面：波紋
雲：流れる雲
粒子：浮遊する光
```

## 具体的な依頼例

### サイバーパンク都市

```
Veo 3.1用の背景画像を生成

【シーン】
- 雨の夜のサイバーパンク都市
- 路地裏、ネオン看板、反射する路面

【構図】
- 奥行きのある一点透視
- 手前に水たまり
- 奥にネオンが輝く

【ライティング】
- ネオンの青とマゼンタ
- 濡れた路面の反射
- 暗い空

【用途】
- スローなドリーインショット
- 8秒かけて路地を進む
```

### 日本の桜並木

```
Veo 3.1用の背景画像を生成

【シーン】
- 満開の桜並木の小道
- 花びらが舞っている

【構図】
- トンネル状の桜の木
- 道が奥に続く
- 花びらが宙を舞う

【ライティング】
- 春の柔らかい日差し
- 木漏れ日
- ピンクと緑のコントラスト

【用途】
- スローパン
- 花びらが流れる様子
```

### 未来のオフィス

```
Veo 3.1用の背景画像を生成

【シーン】
- ハイテクなオープンオフィス
- ホログラフィックディスプレイ

【構図】
- 広角のオフィス全景
- デスク、チェア、植物
- 窓から都市の景色

【ライティング】
- クリーンな人工光
- 青白い蛍光灯
- 窓からの自然光

【用途】
- パンショット
- 未来的な雰囲気
```

### 設計段階

- [ ] 雰囲気が目的に合っている
- [ ] 光の方向が一貫している
- [ ] 動画用の余白・奥行きがある
- [ ] 不要な要素がない
- [ ] 解像度が十分
