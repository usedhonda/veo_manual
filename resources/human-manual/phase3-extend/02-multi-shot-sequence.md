# 複数ショット結合ガイド

## 概要

複数の8秒クリップを結合して、より長く、より複雑な動画を作成する方法を解説します。

**なぜこの方法？**
- Video Extensionの720p制限を回避
- より多様なカメラアングル
- シーン切り替えの柔軟性
- 高品質（1080p）の長尺動画

---

## 複数ショット方式の利点

```
┌─────────────────────────────────────────────────────────────┐
│ Video Extension vs 複数ショット                              │
│                                                             │
│  Video Extension:                                           │
│  ┌────────────────────────────────────────────┐            │
│  │ 8秒 → 16秒 → 24秒 → ...（720p固定）      │            │
│  └────────────────────────────────────────────┘            │
│                                                             │
│  複数ショット:                                              │
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐                              │
│  │8秒 │+│8秒 │+│8秒 │+│8秒 │ = 32秒（1080p可）          │
│  └────┘ └────┘ └────┘ └────┘                              │
│    ↑       ↑       ↑       ↑                               │
│  Shot1   Shot2   Shot3   Shot4                              │
│  各ショットで異なるアングル・構図が可能                      │
└─────────────────────────────────────────────────────────────┘
```

## ワークフロー

### Step 1: ショットリストの作成

```
【例: 製品CM 30秒】

| # | 時間 | ショットタイプ | 内容 | カメラ |
|---|------|---------------|------|--------|
| 1 | 0-8秒 | ヒーロー | 製品フルオービット | orbit |
| 2 | 8-16秒 | ディテール | クローズアップ | static |
| 3 | 16-24秒 | ライフスタイル | 使用シーン | dolly |
| 4 | 24-32秒 | エンディング | ロゴ表示 | static |
```

### Step 2: 一貫性要素の定義

```
【全ショットで統一する要素】
- カラーパレット
- ライティングスタイル
- 製品の見た目
- 音楽/音声のトーン
- テキストスタイル（ある場合）
```

### Step 3: 各ショットの生成

```
Geminiに依頼:
「このショットリストに基づいて、
 各ショットのVeo 3.1プロンプトを生成してください」
```

### Step 4: 編集ソフトで結合

```
【推奨ソフト】
- DaVinci Resolve（無料）
- Adobe Premiere Pro
- Final Cut Pro

【作業内容】
- タイムラインに配置
- トランジション追加
- 音声の調整
- カラー補正（必要に応じて）
```

## 一貫性を保つ技法

### 1. ビジュアルスタイルの統一

```json
{
  "style": {
    "medium": "live_action",
    "color_grade": "teal_orange",
    "film_stock": "ARRI Alexa",
    "lighting": "natural, golden hour"
  }
}
```

全ショットで同じスタイル設定を使用。

### 2. キャラクターの一貫性

```
【Ingredientsモード使用】
- 同じ参照画像を全ショットで使用
- 毎回キャラ特徴を再記述
- ネガティブに「off-model」を追加
```

### 3. カラーパレットの定義

```
【事前に決める】
- 主色: [例: #1a1a2e]
- アクセント: [例: #e94560]
- 補助色: [例: #0f3460]

【プロンプトに含める】
"Color palette: deep navy, coral accent,
teal highlights"
```

### 4. ライティングの統一

```
【全ショットで指定】
- 光の方向: left side key light
- 質: soft diffused
- 色温度: warm (3500K feel)
```

## ショット間の接続

### トランジションの選択

| 状況 | 推奨トランジション | 効果 |
|------|------------------|------|
| 同じシーンの継続 | カット | シームレス |
| 時間経過 | クロスフェード | 滑らか |
| シーン変更 | フェードブラック | 明確な区切り |
| エネルギッシュ | フラッシュ/ワイプ | インパクト |
| サビの頭 | フラッシュ | 強調 |

### 終点→始点の設計

```
【コツ】
Shot 1の終点とShot 2の始点を意識的に設計:

良い例:
Shot 1終点: 人物が右を向く
Shot 2始点: 右から来る別の要素

悪い例:
Shot 1終点: 人物が右を向く
Shot 2始点: 人物が左を向いている（ジャンプカット感）
```

## 音声の扱い

### 音楽の継続性

```
【方法1】後から音楽を追加
- 各ショットは環境音/SFXのみで生成
- 編集時に音楽トラックを追加
- 最も自由度が高い

【方法2】同じ音楽指示
- 全ショットで同じ音楽スタイルを指定
- 編集で音楽のタイミングを調整
```

### 台詞の処理

```
【会話シーンの場合】
- 各ショットに台詞を分割
- または後から音声を差し替え
- リップシンクが必要な部分は慎重に
```

## 品質チェックリスト

### 各ショット生成後

- [ ] スタイルが統一されているか
- [ ] カラーが一貫しているか
- [ ] キャラ/被写体が同じに見えるか
- [ ] ライティングが統一されているか

### 結合前

- [ ] 全ショットの解像度が同じか
- [ ] フレームレートが統一されているか
- [ ] アスペクト比が同じか

### 結合後

- [ ] トランジションは自然か
- [ ] 音声は滑らかに繋がるか
- [ ] 全体のペースは適切か
- [ ] ストーリーの流れは明確か
