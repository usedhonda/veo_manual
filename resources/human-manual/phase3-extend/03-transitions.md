# トランジション技法ガイド

## 概要

Veo 3.1で生成した動画を繋ぐ際のトランジション技法を解説します。

**2つのアプローチ**:
1. Veo 3.1内でトランジションを指示
2. 外部編集ソフトでトランジションを追加

---

## Veo 3.1でのトランジション指示

### 信頼性の高いトランジション

| トランジション | 指示方法 | 信頼性 |
|---------------|---------|--------|
| フェードアウト | `fade to black at end` | ✅ 高 |
| フェードイン | `fade in from black` | ✅ 高 |
| ハードカット | `hard cut` / `cut to` | ✅ 高 |
| グリッチ | `glitch transition` | ✅ 高 |
| ズームトランジション | `zoom into transition` | ✅ 高 |
| フラッシュ | `flash to white` | ⚠️ 中 |
| クロスフェード | `crossfade` | ⚠️ 中 |
| ワイプ | `wipe left/right` | ⚠️ 中 |

### フェード系

```
【フェードアウト】
"...scene fades to black at end"
"...gradually fade to black over final 2 seconds"

【フェードイン】
"Fade in from black, revealing..."
"Start with black screen, fade in to..."

【フェードスルーホワイト】
"Flash to white, then reveal..."
```

### カット系

```
【ハードカット】
"Cut to [次のシーン]"
"Hard cut to..."

【ジャンプカット】
"Jump cut within scene"
※注意: 意図的な場合のみ使用
```

### エフェクト系

```
【グリッチ】
"Glitch transition effect"
"Digital glitch distortion"

【ズーム】
"Zoom into subject, transition to next scene"
"Rapid zoom in transition"
```

## 外部編集ソフトでのトランジション

### 基本トランジション

| トランジション | 用途 | 効果 |
|---------------|------|------|
| クロスディゾルブ | シーン切り替え | 滑らか |
| ディップトゥブラック | 大きな区切り | 明確 |
| ディップトゥホワイト | インパクト | 強調 |
| ワイプ | エネルギッシュ | 勢い |
| スライド | モダン | クリーン |

### 設定ガイドライン

```
【デュレーション推奨値】
- クロスディゾルブ: 0.5-1.5秒
- ディップトゥブラック: 0.5-1.0秒
- ワイプ: 0.3-0.5秒
- フラッシュ: 0.1-0.3秒

【注意点】
- 長すぎると間延びする
- 短すぎると視認できない
- 音楽のビートに合わせる
```

## 状況別の推奨トランジション

### 同じシーンの継続

```
【推奨】ストレートカット（カットのみ）
【理由】同じシーン内では余計なエフェクト不要
【例】会話シーンの切り返し
```

### シーンの変化

```
【推奨】クロスディゾルブ or ディップトゥブラック
【理由】場所/時間の変化を示す
【例】家から学校への移動
```

### 時間経過

```
【推奨】フェードアウト→フェードイン
【理由】時間の流れを明確に
【例】朝から夜への変化
```

### エネルギッシュな場面

```
【推奨】フラッシュ、ワイプ、グリッチ
【理由】勢いとエネルギーを維持
【例】MVのサビ、アクションシーン
```

### 感動的な場面

```
【推奨】スローなクロスディゾルブ
【理由】感情の余韻を残す
【例】再会シーン、クライマックス
```

## ビートとトランジション

### 音楽に合わせる

```
【基本原則】
- トランジションはビートの頭に合わせる
- 強いビートには強いトランジション
- 弱いビートにはソフトなトランジション

【例: 128 BPM】
1ビート = 約0.47秒
4ビート（1小節）= 約1.88秒

トランジションポイント:
- ビートの頭（強調）
- 小節の変わり目（自然）
```

### MVでの使い分け

```
【イントロ】
ゆっくりとしたフェードイン

【Aメロ】
ソフトなカット、クロスディゾルブ

【Bメロ】
少し速いカット

【サビ】
フラッシュ、ワイプ、ビートに合わせたカット

【間奏】
抽象的なトランジション、グリッチ

【アウトロ】
フェードアウト
```

## トランジション連携の設計

### ショット終点→トランジション→ショット始点

```
【良い連携】

Shot 1終点: 人物が右に歩く
  ↓（カット）
Shot 2始点: 新しい場所を右から歩いてくる
→ 動きの方向が一致、自然な流れ

Shot 1終点: 明るい画面
  ↓（フェードアウト）
Shot 2始点: 暗い画面からフェードイン
→ 明→暗→明のリズム
```

```
【避けるべき連携】

Shot 1終点: 人物が右を向く
  ↓（カット）
Shot 2始点: 人物が左を向いている
→ ジャンプカット感、視線が混乱

Shot 1終点: 動きのある画面
  ↓（長いフェード）
Shot 2始点: 動きのある画面
→ フェード中に動きが重なり混乱
```

## Veo 3.1で生成時の考慮

### トランジション用の余白

```
【フェードアウトの場合】
最後の1-2秒を静かな状態に
激しい動きの途中でフェードしない

【フェードインの場合】
最初の1-2秒はシンプルな構図から
複雑な動きは少し待ってから
```

### 動きの継続性

```
【考慮点】
Shot 1の動きの方向
Shot 2で動きをどう受けるか
カメラの動きの整合性
```

## トラブルシューティング

### 問題: トランジションが唐突

```
【対策】
- デュレーションを長くする
- 前後のショットの終点/始点を調整
- イーズイン/イーズアウトを追加
```

### 問題: トランジションが遅い

```
【対策】
- デュレーションを短くする
- よりシャープなトランジションに変更
- 音楽のテンポに合わせる
```

### 問題: 視覚的に混乱

```
【対策】
- シンプルなトランジションに変更
- 前後のショットの明度を調整
- 動きを落ち着かせてからトランジション
```

- [ ] 意図した効果が出ているか
