# ブランド動画制作ガイド

## 概要

企業やブランドのアイデンティティを表現する動画を作成するためのガイドです。

**対応するコンテンツ**:
- ブランドロゴアニメーション
- コーポレートビデオ
- ブランドストーリー
- SNS用ブランドコンテンツ

---

## ブランド動画の種類

### 1. ロゴアニメーション

```
【用途】
- 動画のオープニング/エンディング
- SNSアイコン
- プレゼンテーション

【時間】
- 短め: 3-5秒
- 標準: 5-8秒
```

### 2. ブランドムービー

```
【用途】
- 企業紹介
- 採用動画
- ブランドコンセプト紹介

【時間】
- 30秒〜3分
```

### 3. SNS用ショート

```
【用途】
- Instagram Reels
- TikTok
- YouTube Shorts

【時間】
- 8-15秒
```

## ブランド要素の定義

### カラーパレット

```
【Geminiに伝える情報】
ブランドカラー:
- プライマリ: [HEX] - 使用シーン
- セカンダリ: [HEX] - 使用シーン
- アクセント: [HEX] - 使用シーン
- 背景: [HEX]

例:
- プライマリ: #1a73e8 (Google Blue) - メイン要素
- セカンダリ: #34a853 (Green) - アクセント
- ダーク: #202124 - 背景
```

### トーン&ムード

```
【ブランドパーソナリティ】
- フォーマル vs カジュアル
- テクノロジー vs オーガニック
- ミニマル vs マキシマル
- クラシック vs モダン

例:
「プロフェッショナルだが親しみやすい、
 テクノロジー企業らしいクリーンさ、
 モダンでミニマルなデザイン」
```

### 動きのスタイル

```
【ブランドに合った動き】
高級ブランド:
- ゆっくり、エレガント
- スムーズなトランジション

テック企業:
- シャープ、精密
- 幾何学的な動き

カジュアルブランド:
- 活発、エネルギッシュ
- 遊び心のある動き
```

## 具体的な依頼例

### ロゴアニメーション

```
ブランドロゴのアニメーションを生成

【ブランド情報】
- 業種: テクノロジー企業
- カラー: 青 (#1a73e8) と白
- トーン: モダン、クリーン、プロフェッショナル

【アニメーション】
- スタイル: 幾何学的な粒子が収束
- 動き: 粒子が集まってロゴの形を形成
- スピード: 中程度、精密な動き
- 背景: ダークグレーのグラデーション

【技術】
- duration: 8秒（後で短くカット可能）
- アスペクト比: 16:9
- 解像度: 1080p

【オーディオ】
- SE: デジタルな収束音、満足感のある完成音
- BGM: なし
```

### 抽象ブランドビジュアル

```
ブランドイメージの抽象映像を生成

【ブランド】
- 業種: 金融サービス
- 価値: 信頼、成長、革新
- カラー: ネイビー、ゴールド、白

【ビジュアル】
- 抽象的な線と光
- 上昇する動き（成長を象徴）
- ネットワークのような接続
- 光の粒子が流れる

【カメラ】
- スローなドリーアップ
- 光に向かって上昇

【雰囲気】
- プロフェッショナル
- 信頼感
- 未来志向

【オーディオ】
- BGM: オーケストラ、壮大、希望に満ちた
```

### コーポレートライフスタイル

```
企業の雰囲気を表す映像を生成

【シーン】
- モダンなオフィス空間
- 自然光が差し込む
- 植物とミニマルなデザイン

【人物】
- 多様性のあるチーム
- 協力して働いている
- ポジティブな表情

【カメラ】
- ゆっくりとしたトラッキング
- オフィスを横切る

【カラー】
- ブランドカラーをアクセントに
- 全体は明るく清潔

【雰囲気】
- コラボレーティブ
- イノベーティブ
- ウェルカミング

【オーディオ】
- アンビエンス: オフィスの静かな活気
- BGM: 軽やかなアコースティック
```

## スタイルの一貫性

### 全ショット共通設定

```json
{
  "style": {
    "brand_colors": ["#1a73e8", "#34a853", "#202124"],
    "mood": "professional, modern, clean",
    "lighting": "soft, natural, bright",
    "movement": "smooth, deliberate, elegant"
  },
  "negative": {
    "exclude": [
      "harsh lighting",
      "cluttered composition",
      "dated aesthetics",
      "off-brand colors",
      "chaotic movement"
    ]
  }
}
```

### カラーグレーディング指示

```
【プロンプトに含める】
"Color grade: clean, slightly desaturated,
brand blue (#1a73e8) as accent color,
neutral whites, subtle contrast"
```

## プラットフォーム別の考慮

### YouTube/Vimeo (16:9)

```
【仕様】
- アスペクト比: 16:9
- 解像度: 1080p
- 推奨長さ: 30秒〜3分

【特徴】
- シネマティックな品質
- 詳細なストーリーテリング
- 音声品質重視
```

### Instagram Reels/TikTok (9:16)

```
【仕様】
- アスペクト比: 9:16
- 解像度: 1080p
- 推奨長さ: 8-15秒

【特徴】
- 縦型構図
- 最初の1秒でアテンション
- ループ対応を考慮
```

### LinkedIn/Twitter (16:9 or 1:1)

```
【仕様】
- アスペクト比: 16:9（1:1は非対応）
- 解像度: 1080p
- 推奨長さ: 15-30秒

【特徴】
- プロフェッショナルなトーン
- 字幕があると良い（別途追加）
- クリーンでシンプル
```

## ブランド動画のチェックポイント

### ブランドガイドライン準拠

- [ ] カラーはブランドカラーを使用
- [ ] トーンはブランドパーソナリティに合致
- [ ] メッセージはブランドの価値観を反映
- [ ] 全体の印象がブランドイメージと一致

### 技術的品質

- [ ] 解像度は十分か
- [ ] 動きはスムーズか
- [ ] 色は正確に再現されているか
- [ ] 音声は適切か

### プラットフォーム最適化

- [ ] 正しいアスペクト比
- [ ] 適切な長さ
- [ ] プラットフォームのガイドライン準拠

## トラブルシューティング

### 問題: ブランドカラーが正確でない

```
【対策】
- HEXコードを明示的に指定
- 「exact brand color」を強調
- カラーグレーディング指示を追加
```

### 問題: トーンがブランドに合わない

```
【対策】
- より詳細なムード記述
- 参考ブランドを挙げる
- ネガティブで避けたい要素を指定
```

### 問題: 一貫性がない

```
【対策】
- 全ショットで同じスタイル設定を使用
- テンプレートを作成して再利用
- 編集時にカラー補正で統一
```
