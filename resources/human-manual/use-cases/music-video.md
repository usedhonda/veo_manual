# ミュージックビデオ制作ガイド

## 概要

このガイドでは、Veo 3.1を使ったMV制作のワークフローを解説します。

**対応するMVタイプ**:
- パフォーマンス映像（歌唱、演奏、ダンス）
- ナラティブ（ストーリー性のある映像）
- 抽象・アート映像

---

## MV制作の全体フロー

```
┌─────────────────────────────────────────────────────┐
│ 1. コンセプト: 曲の世界観を映像に変換              │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│ 2. ショットリスト: 8秒クリップを分解               │
│    └→ 3分の曲 = 約22〜23クリップ                   │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│ 3. 各ショットを生成                                 │
│    └→ パフォーマンス / ナラティブ / 抽象           │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────────────────────────────────────────────────┐
│ 4. 編集: クリップを繋いで音楽と同期                │
│    └→ 外部編集ソフト使用                           │
└─────────────────────────────────────────────────────┘
```

## Step 1: コンセプト策定

### 曲の分析

Geminiに伝える情報:
```
【楽曲情報】
- ジャンル: [ロック / ポップ / エレクトロニック / バラード]
- テンポ: [BPM]
- 雰囲気: [感情・ムード]
- 歌詞のテーマ: [主題]

【ビジュアルコンセプト】
- イメージする世界観: [具体的に]
- 参考にしたいMV: [あれば]
- カラーパレット: [色の方向性]
```

### 例: エレクトロニック楽曲

```
【楽曲情報】
- ジャンル: エレクトロニック / Future Bass
- テンポ: 128 BPM
- 雰囲気: 浮遊感、幻想的、少し切ない
- 歌詞: 失われた記憶を探す旅

【ビジュアルコンセプト】
- 世界観: ネオン輝くサイバー都市と自然が融合した異世界
- 参考: Blade Runner 2049 + 新海誠作品
- カラー: シアン、マゼンタ、深い青、時折の暖色アクセント
```

## Step 2: ショットリストの作成

### 8秒クリップへの分解

```
3分の曲 = 180秒 ÷ 8秒 = 約22〜23クリップ

【構成例】
イントロ (0:00-0:16): 2クリップ - 世界観の提示
Aメロ (0:16-0:48): 4クリップ - 主人公の日常
Bメロ (0:48-1:12): 3クリップ - 変化の予兆
サビ (1:12-1:44): 4クリップ - 感情の爆発、ダイナミック
間奏 (1:44-2:00): 2クリップ - 抽象映像
2番 (2:00-2:32): 4クリップ - ストーリー展開
ラスサビ (2:32-3:00): 3-4クリップ - クライマックス
```

### ショットリストのテンプレート

| # | 時間 | タイプ | 内容 | カメラ | 雰囲気 |
|---|------|--------|------|--------|--------|
| 1 | 0:00-0:08 | 抽象 | 粒子が収束 | static | 神秘的 |
| 2 | 0:08-0:16 | 風景 | ネオン都市 | drone | 壮大 |
| 3 | 0:16-0:24 | パフォ | 歌唱（サビ）| close | 感情的 |
| ... | ... | ... | ... | ... | ... |

## 具体的な生成例

### イントロショット（抽象）

```
MV用イントロの抽象映像を生成

【ビジュアル】
- 深い黒の空間に浮かぶ光の粒子
- 粒子がゆっくり収束して人型のシルエットに
- 色: シアンとマゼンタ、白い光点

【動き】
- 非常にゆっくり
- 粒子が螺旋を描きながら集まる
- 最後に一瞬の発光

【音楽】
- エレクトロニック、アンビエントなイントロ
- ゆっくりビルドアップ
```

### サビショット（パフォーマンス）

```
MV用サビのパフォーマンスショットを生成

【アーティスト】
- 女性シンガー、20代
- 衣装: 光沢のある黒のドレス、銀のアクセサリー
- 表情: 感情的、力強い

【シーン】
- ロケーション: 暗いステージ、青と紫のライト
- カメラ: クローズアップ、スローなドリーイン
- ライティング: ドラマチック、顔に陰影

【演技】
- 歌唱（サビの盛り上がり部分）
- 歌詞: "I'm breaking free tonight, nothing holding me back"
- エネルギー: 高い、解放感

【音声】
- 歌詞のリップシンク
- SE: エコー効果
```

### 間奏ショット（幾何学抽象）

```
MV用間奏の抽象映像を生成

【ビジュアル】
- 3D幾何学形状（キューブ、三角形、球体）
- ネオンカラーでエッジが光る
- ビートに合わせてパルス

【動き】
- ビートに同期して形状が回転・変形
- 128BPMのリズムに合わせる
- ドロップ時に一瞬フリーズ→爆発的展開

【カラー】
- シアン、マゼンタ、白
- 黒背景、ハイコントラスト
```

## シームレスループの活用

### 背景ループ素材の作成

```
MVの背景用にシームレスループ映像を生成

【方法】
- Interpolationモードを使用
- 同じ画像をFirst FrameとLast Frameに指定
- 自然にループする動きを指示

【例】
- 炎が燃え続けるループ
- 粒子が漂うループ
- ネオンサインが点滅するループ
```

## 編集のヒント

### クリップの繋ぎ方

| トランジション | 使い所 | 効果 |
|--------------|--------|------|
| ハードカット | ビートに合わせて | エネルギー |
| クロスフェード | ムードチェンジ | 滑らか |
| フラッシュ | サビ頭 | インパクト |
| ワイプパン | セクション変わり目 | 勢い |

### BPMと編集タイミング

```
128 BPM の場合:
- 1ビート = 0.47秒
- 4ビート（1小節）= 1.88秒
- 8秒 = 約4.3小節

カットのタイミング:
- ビートの頭（強調）
- ビート間（流れ）
- 歌詞の切れ目（意味）
```
