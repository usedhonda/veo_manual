# Grok Imagine Lighting & Color Keywords

## Overview

Lighting is crucial for establishing mood and visual quality. Aurora engine simulates light behavior including volumetric effects and subsurface scattering.

---

## Lighting Types

| Keyword | Effect | Best For |
|---------|--------|----------|
| **Volumetric Lighting** | Visible light rays (God rays) | Atmosphere, drama |
| **Rim Light** | Edge highlight separating subject from background | Portraits, products |
| **Subsurface Scattering** | Translucent glow (skin, wax, leaves) | Realistic skin, organic materials |
| **Bioluminescence** | Organic blue/green glow | Sci-fi, fantasy |
| **3-Point Lighting** | Standard studio setup | Products, interviews |
| **Chiaroscuro** | Deep contrast, dramatic shadows | Horror, noir |
| **Caustics** | Light patterns through water/glass | Products, underwater |

---

| Keyword | Effect |

| **Backlit** | Subject silhouette, rim glow |
| **Side Lit** | Dramatic half-face shadow |

## Practical Examples

### Dramatic Portrait
```
Lighting: Single harsh spotlight from above, deep chiaroscuro shadows, rim light on hair edges
```

### Product Shot
```
Lighting: Studio 3-point lighting, soft shadows, rim light highlighting gold edges, sparkling caustics on glass
```

### Cyberpunk Scene
```
Lighting: Neon reflections on wet pavement, volumetric fog, holographic glow, teal and orange color grade
```

### Horror Atmosphere
```
Lighting: Strobe effect, deep shadows, volumetric dust particles, low frequency flickering
```

## Lighting + Camera Combinations

| Mood | Lighting | Camera |
|------|----------|--------|
| Romantic | Golden hour, backlit | Slow dolly, shallow DOF |
| Thriller | Chiaroscuro, strobe | Handheld, rack focus |
| Epic | Volumetric rays, rim light | Wide crane shot |
| Intimate | Candlelight, warm | Close-up, static |
| Futuristic | Neon, bioluminescence | Tracking, FPV drone |
