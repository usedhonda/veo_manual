# Grok Imagine Styles & Aesthetics Keywords

## Overview

Style keywords define the visual treatment and artistic direction. Aurora engine responds well to specific equipment names, film references, and artistic movements.

## Style Combinations (Tested)

### Cinematic Realism
```
Style: 8k photorealistic, Arri Alexa footage, cinematic color grading, film grain, shallow depth of field
```

### Retro Horror
```
Style: VHS glitch, tracking noise, desaturated, found footage aesthetic, 4:3 aspect ratio
```

### Sci-Fi Epic
```
Style: Blade Runner 2049 aesthetic, Unreal Engine 5 render, volumetric lighting, teal and orange
```

### Documentary
```
Style: 16mm film, handheld, natural lighting, muted colors, intimate framing
```

## Warning: Style Conflicts

Avoid mixing incompatible styles:
- ❌ "anime + photorealistic + watercolor"
- ❌ "VHS aesthetic + 8K clean digital"
- ❌ "Ghibli + German Expressionism"

Choose ONE primary style or compatible combinations:
- ✅ "Cinematic anime with soft lighting"
- ✅ "Film noir with modern color grade"
- ✅ "Documentary style with film grain"
