## ワークフロー詳細

### A. 製品CM/広告ルート

**目的**: 商品を魅力的に見せる動画

**フロー**:
```
1. Nano Banana Pro で製品画像を生成
   └→ templates/product-shot.json を使用

2. Veo 3.1 Image-to-Video で動画化
   └→ templates/image-to-video/product-animation.json を使用

3. （オプション）Video Extension で長尺化
```

**Geminiへの依頼例**:
```
高級イヤホンの製品CM動画を作りたいです。

Phase 1: まず製品の静止画を生成してください
- 黒背景、リムライト
- 45度の角度
- プレミアム感重視

Phase 2: その画像を元に動画プロンプトを生成
- スローなオービット
- 8秒
- モダンエレクトロニックBGM
```

---

### B. アニメキャラクタールート

**目的**: キャラクターの一貫性を保った動画

**フロー**:
```
1. Nano Banana Pro でキャラクターシートを生成
   └→ templates/character-sheet.json を使用

2. Veo 3.1 Ingredients モードで動画生成
   └→ templates/ingredients/character-scene.json を使用

3. （オプション）複数シーンを連続生成
   └→ 同じ参照画像を使い回し
```

**Geminiへの依頼例**:
```
アニメキャラクターのMVを作りたいです。

キャラ設定:
- 青い髪のツインテール
- 赤い瞳
- アイドル衣装（白とピンク）

Phase 1: このキャラのシートを生成
Phase 2: ステージで踊るシーンの動画プロンプトを生成
        - Ingredientsモード用
        - エネルギッシュ、アイドルアニメ風
```

---

### C. 映像表現ルート（MV/抽象）

**目的**: アーティスティックな映像表現

**フロー**:
```
Text-to-Video: 自由な映像表現
または
Interpolation: 2つの画像間の変化

Phase 1は不要な場合が多い
```

**Geminiへの依頼例**:
```
エレクトロニック楽曲のMV用抽象映像を作りたい

- 幾何学的な形状がビートに合わせて動く
- ネオンカラー（シアン、マゼンタ）
- 黒背景、ハイコントラスト
- シームレスループ可能だとベター
```

## モード選択チャート

### 何を使うべき？

| あなたの状況 | 推奨モード |
|-------------|-----------|
| 特定の画像からスタートしたい | Image-to-Video |
| キャラを複数シーンで使いたい | Ingredients |
| AからBへの変化を見せたい | Interpolation |
| 特に制約なし、自由に作りたい | Text-to-Video |
| 既存の動画を延長したい | Video Extension |

### 解像度の選び方

| 目的 | 解像度 |
|------|--------|
| 最終納品、クライアント提出 | 1080p |
| プロトタイプ、テスト | 720p (fast) |
| 動画延長が必要 | 720p（制約） |

### アスペクト比の選び方

| 用途 | 比率 |
|------|------|
| YouTube, テレビ, 映画 | 16:9 |
| TikTok, Instagram Reels, Shorts | 9:16 |

## Phase 1 → Phase 2 の連携

### 画像最適化のポイント

1. **解像度を合わせる**: 出力動画と同じ解像度で画像生成
2. **余白を考慮**: 動きが入る方向に余白を確保
3. **一貫性アンカー**: 色、ライティング、スタイルを明記

### 連携時の依頼例

```
【Phase 1リクエスト】
以下の仕様でFirst Frame用画像プロンプトを生成:
- 製品: スマートウォッチ
- 構図: 左寄せ（右にオービット用の余白）
- ライティング: 右からのリムライト
- 解像度: 1080p, 16:9

【Phase 2リクエスト】
上記画像を使ってImage-to-Video用プロンプトを生成:
- アクション: 右方向へのスローオービット
- 一貫性維持: 同じリムライト、背景
- 音声: プレミアム感のあるエレクトロニック
```
