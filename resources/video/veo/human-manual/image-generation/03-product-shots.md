# 製品撮影用画像ガイド

## 概要

製品CMやプロモーション動画のためのFirst Frame画像を生成するガイドです。

**目的**:
- 高級感のある製品画像
- Veo 3.1でのオービット/ドリー動画用
- ブランドイメージに合った撮影

---

## 製品撮影の基本

### プロダクトフォトの種類

| 種類 | 用途 | 特徴 |
|------|------|------|
| ヒーローショット | メインビジュアル | 製品の魅力を最大限に |
| ディテール | 質感・機能強調 | マクロ、部分アップ |
| ライフスタイル | 使用シーン | 環境との調和 |
| グループ | 商品ラインナップ | 複数製品の配置 |

### 動画用の考慮点

```
┌─────────────────────────────────────────────────────────────┐
│ 動画での動きを考慮した構図                                  │
│                                                             │
│ オービット用:                                               │
│   └→ 製品の周囲に空間、360度対応の配置                     │
│                                                             │
│ ドリーイン用:                                               │
│   └→ 製品をやや奥に、前方に空間                            │
│                                                             │
│ ライトスイープ用:                                           │
│   └→ 反射面を意識、ハイライトが映える角度                  │
└─────────────────────────────────────────────────────────────┘
```

## 製品カテゴリ別ガイド

### テクノロジー製品

```
【製品例】
イヤホン、スマートフォン、ノートPC、スマートウォッチ

【ライティング】
- リムライト（縁取り）が効果的
- クールな色温度
- ハードなライト、クリーンな影

【背景】
- ダークグラデーション（黒〜チャコール）
- または白の無限背景
- 反射面（アクリル板上）

【構図】
- 45度の俯瞰
- 製品の特徴が見える角度
- ロゴは控えめに
```

**依頼例**:
```
テクノロジー製品のFirst Frame画像を生成

【製品】
- ワイヤレスイヤホン（マットブラック、ローズゴールドアクセント）
- 充電ケースと一緒に

【撮影仕様】
- 構図: 45度俯瞰、ケース開いた状態
- 背景: チャコールグラデーション
- ライティング: バック右からのリムライト
- 反射: 下に微かな反射面

【動画用設計】
- オービット用の周囲スペース
- 8秒で180度回転を想定
```

### 美容・コスメ製品

```
【製品例】
スキンケア、香水、メイクアップ、ヘアケア

【ライティング】
- ソフトな自然光風
- ゴールデンアワーの暖かさ
- 製品の質感を強調

【背景】
- パステル系グラデーション
- 大理石、木目などのテクスチャ
- 花やリーフなどの装飾

【構図】
- ボトルの曲線を活かす角度
- ドロップやテクスチャの表現
- エレガントな配置
```

**依頼例**:
```
美容製品のFirst Frame画像を生成

【製品】
- フェイスセラム（琥珀色ガラス瓶、ゴールドキャップ）
- ドロッパー付き

【撮影仕様】
- 構図: 中央やや左、斜め45度
- 背景: ソフトピーチのグラデーション
- ライティング: 左上からの自然光風
- 小物: ゴールドの葉、白い花びら散らし

【動画用設計】
- スローなドリーイン用
- ドロッパーから滴が落ちる予定
```

### 食品・飲料

```
【製品例】
飲料ボトル、食品パッケージ、調理シーン

【ライティング】
- サイドライト（テクスチャ強調）
- バックライト（透明感）
- 暖色系の食欲をそそる光

【背景】
- 木目やキッチンテーブル
- 食材と一緒に配置
- シンプルな単色

【構図】
- 製品を主役に
- 食材は補助的に
- 湯気や水滴の表現
```

**依頼例**:
```
飲料製品のFirst Frame画像を生成

【製品】
- クラフトビール瓶（アンバーガラス）
- 水滴がついた冷えた状態

【撮影仕様】
- 構図: 中央、やや低いアングル
- 背景: ダークウッドテーブル、ぼかした居酒屋風
- ライティング: バックライトで透過光
- 効果: 水滴、霜、ゴールドの輝き

【動画用設計】
- クレーンダウンショット用
- 栓を開ける前の状態
```

### 高級時計・宝飾品

```
【製品例】
腕時計、ネックレス、指輪、高級ペン

【ライティング】
- ドラマチックなサイドライト
- ピンポイントのハイライト
- 深い影でコントラスト

【背景】
- リッチなベルベット
- ダークで深みのある色
- シンプルで上品

【構図】
- 極めてクローズアップ
- 素材の質感を見せる
- 光の反射を計算
```

**依頼例**:
```
高級時計のFirst Frame画像を生成

【製品】
- メンズ腕時計（シルバーケース、黒文字盤）
- クロノグラフ機能

【撮影仕様】
- 構図: 文字盤45度、マクロ風
- 背景: ディープネイビーのグラデーション
- ライティング: 左からの強いサイドライト
- 効果: 針と数字に細かいハイライト

【動画用設計】
- 微動オービット（わずかな動き）
- 光が文字盤を横切る
```

## 共通の技術仕様

### 解像度とアスペクト比

```
【Veo 3.1互換】
- アスペクト比: 16:9（横長）または 9:16（縦長）
- 解像度: 1080p相当以上
- 高解像度で生成→Veoが自動調整
```

### ライティングの指定

```
【方向】
- front: 正面から
- side: 横から（左/右）
- back: 後ろから（リムライト）
- top: 上から

【質】
- hard: くっきりした影
- soft: 柔らかい影
- diffused: 拡散光

【色温度】
- warm: 暖色（3000K風）
- neutral: 自然（5500K風）
- cool: 寒色（7000K風）
```

## 動画を意識した構図設計

### オービット用

```
【要件】
- 製品の全周に空間
- どの角度からも見栄えする配置
- 背景は均一またはグラデーション

【構図ルール】
- 製品を中央に
- 周囲にフレームの30%以上の空間
- 影は控えめに
```

### ドリーイン用

```
【要件】
- 前方に十分な空間
- 製品はやや遠くに
- 焦点が当たる部分を考慮

【構図ルール】
- 製品を三分割の奥側
- 前景に空間またはボケ要素
- 深度感のある配置
```

### ライトスイープ用

```
【要件】
- 反射面を持つ製品
- ハイライトが動くための表面
- ライトの軌道を想定

【構図ルール】
- 反射面が見える角度
- ライトの通り道を確保
- コントラストの高い設定
```
