# アニメキャラクター動画ガイド

## 概要

このガイドでは、アニメキャラクターの一貫性を保ちながら複数のシーンを生成する方法を解説します。

**完成イメージ**:
- 同じキャラクターが複数シーンに登場
- キャラデザインの一貫性を維持
- MVやショートアニメに使える品質

## Step 1: キャラクター設計

### 必須要素を決める

Geminiに伝える情報:

```
【基本情報】
- 名前: ミク（仮）
- 年齢/見た目: 16歳くらいの少女
- 性格: 元気、明るい、ちょっとドジ

【ビジュアル設計】
髪:
- 色: ターコイズブルー
- スタイル: ツインテール、長め
- 特徴: 黒いリボンで結んでいる

目:
- 色: 同じターコイズブルー
- 形: 大きめ、丸みのあるアニメ目

服装:
- ベース: 学校制服（セーラー服）
- カラー: 白ブラウス、紺のスカート
- 特徴: 赤いリボンタイ、ニーハイソックス

アクセサリー:
- ヘアリボン（黒）
- 学生カバン
```

### アニメスタイルを決める

| スタイル | 特徴 | 適したシーン |
|---------|------|-------------|
| 少年漫画風 | ダイナミック、影強め | アクション |
| 少女漫画風 | ソフト、キラキラ | ロマンス |
| 日常アニメ風 | クリーン、明るい | コメディ |
| 新海誠風 | 背景リアル、光の表現 | 風景重視 |
| ユーフォテーブル風 | エフェクト重視 | バトル |

## Step 2: Phase 1 - キャラクターシート生成

### Geminiへの依頼

```
Veo 3.1のIngredients用キャラクターシートを生成してください。

【キャラ情報】
- 髪: ターコイズブルーのツインテール
- 目: 同じターコイズブルー
- 服: セーラー服（白ブラウス、紺スカート、赤リボン）
- アクセサリー: 黒いヘアリボン

【シート要件】
- 複数アングル表示（正面、斜め45度、横顔）
- 表情バリエーション（笑顔、真剣、驚き）
- 背景: シンプルな白またはグレー
- スタイル: クリーンなアニメ風、modern anime

【技術仕様】
- 解像度: 1080p相当
- 16:9の比率
- Ingredients用の参照画像として使用予定
```

### 出力されるシート構成例

```
┌─────────────────────────────────────────────┐
│  [正面]    [45度]    [横顔]    [後ろ]       │
│                                             │
│  [笑顔]    [真剣]    [驚き]    [照れ]       │
│                                             │
│  カラーパレット: #00CED1, #000080, #FF0000  │
└─────────────────────────────────────────────┘
```

```

- シーン: [具体的なシーン説明]
- アクション: [キャラの行動]
- 雰囲気: [感情・ムード]

【一貫性の維持】
必ず以下を維持:
- 髪色: ターコイズブルー
- 髪型: ツインテール + 黒リボン
- 服装: セーラー服（白、紺、赤リボン）
- 目の色: ターコイズブルー

【スタイル】
- medium: anime
- カラー: [シーンに合った色調]
```

## 複数シーンの一貫性を保つコツ

### 1. 参照画像は同じものを使い回す

```
全てのシーンで同じキャラクターシートを
referenceImageとして指定する
```

### 2. 毎回キャラ特徴を再記述

プロンプトに必ず含める:
```json
"subject": {
  "reference": "Character from reference image",
  "appearance": {
    "reinforce": "Turquoise blue twin-tail hair with black ribbons, turquoise eyes, sailor uniform (white, navy, red bow)",
    "consistent": "Hair color, hairstyle, eye color, uniform design"
  }
}
```

### 3. ネガティブプロンプトに追加

```json
"negative": {
  "exclude": [
    "off-model",
    "wrong hair color",
    "different outfit",
    "different hairstyle",
    "realistic style",
    "3D render",
    "morphing"
  ]
}
```

### 4. スタイルを統一

全シーンで同じ指定:
```json
"style": {
  "medium": "anime",
  "sub_style": "modern anime, clean lines",
  "consistency": "Match character design from reference exactly"
}
```

## よくある問題と対策

### 問題1: 髪の色が変わる

**原因**: 照明やシーンによる色変化

**対策**:
- 具体的な色を再指定: "turquoise blue (hex: #00CED1)"
- ネガティブに追加: "wrong hair color", "color shift"

### 問題2: 服装が変わる

**原因**: 説明が曖昧

**対策**:
- 詳細に記述: "sailor uniform with white blouse, navy pleated skirt, red ribbon tie"
- ネガティブに追加: "different outfit", "costume change"

### 問題3: 顔が安定しない

**原因**: Ingredientsの限界、複雑なシーン

**対策**:
- シンプルなアクションから始める
- `no morphing`を必ず追加
- 顔のアップは避ける（ミディアムショット推奨）

### 問題4: アニメじゃなくなる

**原因**: `anime`スタイルが弱い

**対策**:
- 明示的に指定: `medium: "anime"`, `sub_style: "modern anime"`
- ネガティブに追加: `realistic`, `photorealistic`, `3D render`

## チェックリスト

### シート生成時

- [ ] 複数アングルを含めた
- [ ] カラーパレットが明確
- [ ] 特徴的な要素（髪飾り等）が見える
- [ ] 背景がシンプル

### 動画生成時

- [ ] 参照画像を指定した
- [ ] キャラ特徴を再記述した
- [ ] スタイルを"anime"に指定
- [ ] 一貫性関連のネガティブを追加
- [ ] 複雑すぎないアクションにした
